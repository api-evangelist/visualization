We recently added support for visualizing the OpenShift API in order to build clients against the server.Kubernetes added support for Swagger.io API descriptions of their resources, and we’ve begun enabling that in OpenShift master.To see the API docs, run OpenShift locally: argument, which allows JavaScript to use the server – this will be important in the last step.Since Swagger is a structured API description, you can use the Swagger UI to read the API description and visualize it.
