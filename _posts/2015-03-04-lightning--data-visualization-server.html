Lightning provides a set of web-based visualizations that can plug into your existing workflow.Click on points in the graph to see their connections.The Freeman Lab at Janelia Research Campus uses Lightning to visualize large-scale neural recordings from zebrafish, in collaboration with the Ahrens Lab.Lightning is a data-visualization server providing API-based access to reproducible, web-based, interactive visualizations.It includes a core set of visualization types, but is built for extendability and customization.
